Function withDependency() gets array as an argument, sorts it and uses "underscore" library to return the first and the last elements of the array. We must provide url to the library as "importFiles" value. You can pass dependencies as many as you need. 

	
function withDependency(arr){
      var ret = [];

      function compare(a,b){
        return b.stargazers_count - a.stargazers_count
      }

      arr.sort(compare)
      ret.push(_.first(arr));
      ret.push(_.last(arr));
      return ret;
  }

run it in thred
